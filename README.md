# Run micro-site
Start the client
1. npm install
2. npm run serve

Start the server
1. python3 -m venv env
2. start virtual environment source env/bin/activate
3. pip install -r requirements.txt
4. python app.py

Start mysql database locally
1. create database data
2. create table using sql script
